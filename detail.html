CoVid19 Tracker Map For India (and Subdivisions)

This map displays the number of total confirmed cases (or recovered or deceased as is selected) in India in each State and Union Territories (Updated to date).
1. Hovering over a particular state Shows the name of the state along with the exact Number of reported cases.
2. Clicking on a particular state Changes the following graphs and charts with respect to the state.
3. The Darker the color of a state the severe is the condition of CoVid spread.
